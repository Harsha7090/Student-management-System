# Student-management-System

This is a simple full stack web application developed using Java Spring Boot and MySQL. The application allows users to manage student records with basic CRUD operations.

## About the Project
The main goal of this project is to understand how frontend and backend communicate using REST APIs. It follows a layered architecture (Controller → Service → Repository) for better code organization.

## Features
- Add new student details  
- View all students  
- Delete student records  
- REST API integration between frontend and backend  

## Technologies Used
- Java (Spring Boot)
- MySQL Database
- HTML, CSS, JavaScript
- REST APIs
##  Project Structure
The backend is divided into:
- Controller → Handles API requests  
- Service → Contains business logic  
- Repository → Handles database operations  

Frontend is built using simple HTML and JavaScript to interact with APIs.

## How to Run
1. Install and start MySQL  
2. Create a database named `studentdb`  
3. Update username and password in `application.properties`  
4. Run the Spring Boot application  
5. Open `index.html` in your browser
